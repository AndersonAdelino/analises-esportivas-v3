Nao precisa mais:
  ✗ Fechar o navegador
  ✗ Abrir terminal
  ✗ Rodar scripts Python

  ✓ Aguardar 2-3 minutos

  ✓ Pronto!

┌──────────────────────────────────────────────────────────────────────────────┐
│  COMO USAR                                                                   │
└──────────────────────────────────────────────────────────────────────────────┘

PASSO 1: Verifique o status
  └─> Olhe a cor (verde/amarelo/vermelho)
  └─> Veja a idade dos dados

PASSO 2: Clique no botao "Atualizar Dados Agora"
  └─> Aparecera um spinner: "Coletando dados..."

PASSO 3: Aguarde 2-3 minutos
  └─> O sistema esta buscando dados da API
  └─> 20 times x 20 partidas cada = 400+ jogos

PASSO 4: Veja a confirmacao
  └─> Mensagem: "Dados atualizados! 412 partidas coletadas"
  └─> Baloes aparecem na tela!

PASSO 5: Recarregue a pagina
  └─> Pressione F5
  └─> OU clique em "Limpar Cache" mais abaixo

PASSO 6: Use os dados atualizados!
  └─> Agora suas analises usam dados frescos
  └─> Incluindo jogos de hoje (se ja finalizados)

┌──────────────────────────────────────────────────────────────────────────────┐
│  EXEMPLO PRATICO                                                             │
└──────────────────────────────────────────────────────────────────────────────┘

SITUACAO: Voce quer analisar Botafogo x Mirassol

ANTES:
  1. Dados do Botafogo de 2 dias atras
  2. Predicao baseada em jogos antigos
  3. Jogo de hoje do Botafogo NAO incluido
  4. Analise pode estar imprecisa

SOLUCAO:
  1. Clique em "Atualizar Dados Agora"
  2. Aguarde 2-3 minutos
  3. Recarregue pagina (F5)
  4. Agora tem jogo de hoje!

DEPOIS:
  1. Dados do Botafogo atualizados
  2. Jogo de hoje INCLUIDO
  3. Predicao mais precisa
  4. Aposte com confianca!
